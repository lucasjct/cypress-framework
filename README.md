# cypress-framework

## How to install Cypress

1. `npm init`
2. `npm install cypress --sev-dev`
3. Configure the *pakage.json* script property to: 
```js
{
  "scripts": {
    "cypress:open": "cypress open"
  }
}

4. To run: npm run cypress:open
```
